WTF is this

Graphics information exposed by the WebGL API is unreliable and doesn't allow an approximation of performance. We record the GL renderer name and match it with benchmark results. The more people use the benchmark, the more accurate we'll be.

Quickstart

The library is an npm installable list of WebGL renderers you can use to pinpoint the performance capabilities of someone using your app. You can weight the performance data so it's even more accurate for your particular use case.

The Benchmark

The benchmark tests texture IO, shaders, and computation. We average and weight them together to get a wholistic view of the performance of a particular GL renderer.
