Open Data

We then take that data, normalize it, and dump it into an NPM package. You use it to calibrate your WebGL apps to match the capabilities of your users' devices.

Quickstart

Install the library

$ npm install --save benchmark.gl

Use

const bmgl = require("benchmark.gl");

const performanceQuotient = bmgl.calibrate({
    preset: 'graphics' // or 'compute', 'shader'
});

That's it.
